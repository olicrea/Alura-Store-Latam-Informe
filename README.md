# Alura-Store-Latam-Informe
Este proyecto tiene como objetivo principal asesorar al cliente: Sr. Juan, en la toma de decisiones estratégicas para su cadena de tiendas. El análisis busca identificar la unidad de negocio más adecuada para ser vendida, con el fin de capitalizar un nuevo emprendimiento sin comprometer la estabilidad financiera global de la marca.

Para ello, se realizó un análisis de métricas de cuatro sucursales evaluando:

- Ingresos totales y participación de mercado.
- Rendimiento por categorías y productos líderes.
- Satisfacción del cliente.
- Desempeño logístico (costos de envío) y cobertura geográfica.

## Tecnologías y Herramientas
- Lenguaje: Python.
- Entorno de desarrollo: Google Colab, utilizado para la ejecución de celdas de código de forma interactiva y la visualización inmediata de gráficos.
- Bibliotecas usadas: Pandas y Matplotlib.

## Ejecución del Proyecto
Recomendación: Google Colab. Pasos:
- Sube el archivo .ipynb a tu unidad de Google Drive.
- Ábrelo con "Google Colaboratory".
- Ejecuta las celdas en orden secuencial (Ctrl + F9 para ejecutar todas).

## Estructura del Análisis
1. **Importación y Exploración:** Consolidación de datos de las 4 tiendas en diccionarios para un procesamiento optimizado.
2. **Análisis de Métricas.**
3. **Conclusiones:** Identificación de la tienda recomendada para la desincorporación.
